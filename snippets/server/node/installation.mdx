The Node.js SDK is hosted [here](https://www.npmjs.com/package/statsig-node). You can install the SDK using NPM or Yarn:

<CodeGroup>
```bash npm
npm install statsig-node
```

```bash yarn
yarn add statsig-node
```
</CodeGroup>
