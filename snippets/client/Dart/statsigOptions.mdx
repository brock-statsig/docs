`Statsig.start()` takes an optional parameter `options` in addition to `sdkKey` and `user` that you can provide to customize the Statsig client.

<ResponseField name="initTimeout" type="int" default="3">
  Seconds to wait for the initial request before completing. Returns cached/default values if the request hasn't completed. Set to `0` to wait indefinitely.
</ResponseField>

<ResponseField name="api" type="string" default="https://statsigapi.net/v1/">
  Endpoint for all SDK network requests. Do not override unless you implement the Statsig API elsewhere.
</ResponseField>

<ResponseField name="environment" type="string" default="">
  Environment tier for evaluations ("development", "staging", "production").
  Default empty string behaves like "production". Non-production events are excluded from Pulse results.
</ResponseField>
