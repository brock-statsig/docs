## Checking a Gate

<Tabs>
<Tab title="Swift">
```swift
if Statsig.checkGate("new_homepage_design") {
  // Gate is on, show new home page
} else {
  // Gate is off, show old home page
}
```
</Tab>

<Tab title="Objective-C">
```swift
if ([Statsig checkGateForName:@"new_homepage_design"]) {
  // Gate is on, show new home page
} else {
  // Gate is off, show old home page
}
```
</Tab>
</Tabs>
