## Logging an Event

<Tabs>
<Tab title="Swift">
```swift
Statsig.logEvent(withName: "purchase", value: 2.99, metadata: ["item_name": "remove_ads"])
```
</Tab>

<Tab title="Objective-C">
```swift
[Statsig logEvent:@"purchase" doubleValue:2.99 metadata:@{@"item_name" : @"remove_ads"}];
```
</Tab>
</Tabs>
